最大待ち時間 

最大待ち時間を設定すると、条件を満たすオブジェクトが存在しない場合に、条件を満たすオブジェクトが作成されるまで設定された秒数(最大で30秒)だけサーバが応答を保留します。これにより何度もリクエストしなくてもオブジェクトの作成をそれなりの精度で検出できます。サーバで応答が保留されている間は結果が返らないので、アプリケーションが停止しないように注意してください。
