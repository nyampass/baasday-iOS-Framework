\subsection{Detailed Description}
端末情報を保持するクラスです。 

インスタンスは\-B\-D\-Authenticated\-Userクラスのcurrent\-Deviceプロパティで取得します。更新は\-B\-D\-Authenticated\-Userクラスのupdate\-Deviceメソッドを使用します。

端末情報の取得、更新を行う前に\-B\-D\-Baasdayクラスのset\-Device\-Idメソッドで端末\-I\-Dを設定する必要があります。まだ一度も端末情報を保存していない場合はgenerate\-Device\-Idメソッドで端末\-I\-Dを生成し、それを\-B\-D\-Baasdayクラスに設定します。生成した端末\-I\-Dはアプリケーション内に保存してください。二度目以降は保存した端末\-I\-Dを\-B\-D\-Baasdayクラスに設定して使用します。
