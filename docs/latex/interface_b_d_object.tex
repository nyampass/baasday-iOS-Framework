\subsection{Detailed Description}
baasdayサーバ上に保存されるオブジェクトの共通の基底クラスです。フィールドの値の取得、baasday上のデータの更新、削除の機能を提供します。 

フィールドの値は\-N\-S\-Number(数値/ブール)、\-N\-S\-String(文字列)、\-N\-S\-Date(日付)、\-N\-S\-Array、\-N\-S\-Dictionary、nilで表されます。
