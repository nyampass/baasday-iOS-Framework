Detailed Description

baasdayサーバ上に保存されるオブジェクトの共通の基底クラスです。フィールドの値の取得、baasday上のデータの更新、削除の機能を提供します。

フィールドの値はNSNumber(数値/ブール)、NSString(文字列)、NSDate(日付)、NSArray、NSDictionary、nilで表されます。
